## 📱 Digital Distraction Analyzer Dashboard

A data-driven Streamlit dashboard that analyzes digital behavior patterns such as screen time, notification frequency, and productivity levels to identify digital distractions and their impact on performance.

## 🚀 Project Overview

The Digital Distraction Analyzer helps understand how excessive screen time and notifications affect productivity.
It uses synthetic behavioral data and visual analytics to provide insights into digital habits.

This project demonstrates:

Data Cleaning & Preprocessing

Exploratory Data Analysis (EDA)

Data Visualization

Correlation Analysis

Interactive Dashboard using Streamlit

## 📊 Key Features

📈 Screen Time vs Productivity Analysis

🔔 Notification Distribution Visualization

🔥 Correlation Heatmap

# 📊 Summary Metrics (Average Screen Time, Notifications, Productivity Score)

🖥 Interactive Web Dashboard

## ▶️ How to Run the Project

1️⃣ Clone the repository
git clone https://github.com/omtarade510-hue/Digital_Distraction_Analyzer.git
cd Digital_Distraction_Analyzer

2️⃣ Install dependencies
pip install -r requirements.txt

3️⃣ Run the Streamlit app
streamlit run Dashboard/app.py

Then open:

http://localhost:8501
