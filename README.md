# 📱 Digital Distraction Analyzer

A data analytics project to study the impact of digital behavior on productivity
using synthetic data and interactive dashboards.

## 🚀 Features
- Exploratory Data Analysis (EDA)
- Interactive Streamlit Dashboard
- Filters by age
- Correlation analysis
- Productivity insights

## 🗂 Project Structure
Digital_Distraction_Analyzer/
├── Code/
│ ├── eda.py
│ └── dashboard.py
├── Data/
│ └── digital_behavior.csv
├── requirements.txt
└── README.md
